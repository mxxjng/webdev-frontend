A Quickstart

A1: Miniübung: 5 Buttons

Erstellen Sie eine Web-Seite mit fünf Buttons, die

  • bestimmte Satzteile fett (1. Button) und rot (2. Button) erscheinen lassen
  • ein Bild laden (3. Button) und wieder verschwinden lassen (4. Button)
  • die Schriftgröße des gesamten Textes verändern (5. Button)

A2: Miniübung: Separate Skript-Datei

Lagern Sie die Eventhandler-Funktionen der vorigen Miniübung in eine separate JavaScript-Datei aus.

A3: Wiederholung HTML, CSS

Erstellen Sie eine HTML-Seite (vgl. Bild links) mit

  • einem Titel
  • einer zentrierten, farbigen Überschrift
  • einem Absatz mit "Lorem Ipsum..."-Text, in dem einzelne Wörter farblich hervorgehoben sind.

    "Lorem Ipsum"-Text finden Sie durch eine einfache Internet-Suche. Verwenden Sie ein internes StyleSheet zum hervorheben der einzelnen Wörter und KEIN Inline-StyleSheet. Achten Sie dabei darauf, dass sich die farbliche Hervorhebung möglichst einfach für mehrere farblich hervorgehobene Wörter ändern lassen muss.

A4: Wiederholung Tabellen, Bilder

Erstellen Sie unter Verwendung einer HTML-Tabelle eine einfache Personen-Datenbank die folgende Angaben zu einer Person darstellen soll:

  • Bild
  • Name
  • Email

Bilder finden Sie im Verzeichnis der Übung.

  • Verwenden Sie eine Tabelle mit drei Spalten, wie im Bild links dargestellt (zum Vergrößern bitte darauf klicken).
  • Sämtliche Daten kodieren Sie bitte direkt in HTML. Wie die Daten aus einer Datenbank gelesen werden können, behandeln wir später.
  • Binden Sie Email-Adressen als Link-Element und "mailto" ein.
  • Passen Sie die Höhe der Bilder an.

A5: Copyright-Datum mit Javascript

  • Erstellen Sie eine kleine HTML-Seite mit einer Copyright-Anzeige die automatisch das aktuelle Jahr anzeigt.

    new Date().getFullYear() gibt das aktuelle Jahr zurück.

  • Fügen Sie einen Button hinzu, der bei einem Klick einen alert mit dem aktuellen Datum ausgibt.

A6: Console

Probieren Sie folgende Methoden von console aus:

  • console.log()
  • console.info()
    Übergeben Sie auch mal mehrere Parameter verschiedener Typen.
  • console.warn()
  • console.error()
  • console.group() und console.groupEnd()

A7: Dark-Mode

In der folgenden Aufgabe soll ein Dark-Mode Switch implementiert werden.

  • Führen Sie bei Klick auf die Radio-Inputs eine Methode aus, die dem body die Klasse theme-dark hinzufügt bzw. entfernt.

    document.body.classList besitzt eine add- bzw. remove-Methode.
