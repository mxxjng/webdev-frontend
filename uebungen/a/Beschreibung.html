A5: Copyright-Datum mit Javascript

  • Erstellen Sie eine kleine HTML-Seite mit einer Copyright-Anzeige die automatisch das aktuelle Jahr anzeigt.

    new Date().getFullYear() gibt das aktuelle Jahr zurck.

  • Fgen Sie einen Button hinzu, der bei einem Klick einen alert mit dem aktuellen Datum ausgibt.
