    Es werden noch die Variablen counter und numClicks benötigt, die sich außerhalb des Scopes der Funktion befinden.

B7: while vs. do-while

Beantworten Sie folgende Fragen:

  • Was ist der Unterschied zwischen while und do-while?
  • Überlegen Sie ohne den Code auszuführen, welche Ausgabe dieses Script erzeugt?: 
    let x = 9

while (x < 9 && x > 0) {
    x--
    console.log('x in while-schleife: ' + x)
}
console.log('x nach while-schleife: ' + x)

x = 9
do {
    x--
    console.log('x in do-while-schleife: ' + x)
} while(x < 9 && x > 0)

console.log('x nach do-while-schleife: ' + x)
  • Führen Sie den Code aus und vergleichen Sie das Ergebnis mit Ihrer Überlegung.

B8: Dance Floor, logische Operatoren

In dieser Aufgabe sollen logische JS-Operatoren angewendet werden, um das Interface einer Lichtanlage zu steuern.

    • Öffnen Sie die Datei lesson-03-exercise-08.js in Ihrem Editor.
      Die Variablen am Anfang der Datei halten den Status der Buttons und werden bei der Implementierung der Methoden benötigt.
    • Zeigen Sie die Datei lesson-03-exercise-08.html in Ihrem Browser an.
      Öffnen Sie auch die Developer-Tools um später Fehler schnell zu erkennen.
  2. Stand: Es sollten alle RGB-Felder rot leuchten.
  3. Implementieren Sie die Methode getRandomColor()
    • Die Werte red, green, blue sollen mit gleicher Wahrscheinlichkeit zurückgegeben werden

      Die Methode Math.random() gibt eine Zahl zwischen 0 und 1 (z.B. 0.1) zurück.

    • Ergebnis: Die RGB-Felder sollten nun alle leuchten, aber ihre Farben wechseln
  4. Implementieren Sie die Methode isFieldOn()
    • Die Methode soll false zurückgeben, wenn Black oder Strobe aktiv sind.
    • Die Methode soll mit einer Wahrscheinlichkeit von 20% true zurückgeben, wenn RGB aktiv ist.
    • Ergebnis: Es sollte ein Teil der RGB-Felder in verschiedenen Farben leuchten. Bei Togglen von RGB, Black oder Strobe sollten die Felder entsprechend an- bzw. ausgehen.
  5. Implementieren Sie die Methode isStrobeOn()
    • Die Methode soll false zurückgeben, wenn Black aktiv ist.
    • Die Methode soll true zurückgeben, wenn Strobe aktiv ist.
    • Ergebnis: Der Toggle von Strobe beeinflusst nun den Status der Strobe-Lichter.
  6. Implementieren Sie die Methode isBlinderOn()
    • Die Methode solle false zurückgegeben wenn Black oder Strobe aktiv sind.
    • Die Methode soll true zurückgeben, wenn Blinder aktiv ist.
    • Ergebnis: Bei Strobe oder Black sind die Blinder aus. RGB-Felder und Blinder können unabhängig voneinander aktiv sein.
  7. Bonusaufgabe: Der Random-Schalter soll einen automatischen Modus ermöglichen.
    Implementieren Sie dazu die Methode isRandomOn()
    • Die Methode soll false zurückgeben, wenn Black oder Strobe aktiv sind.
    • Die Methode soll mit einer Wahrscheinlichkeit von 40% true zurückgeben, wenn Random aktiv ist.
    • Erweitern sie die Methoden isFieldOn(), isStrobeOn() und isFieldOn(). Diese sollen nicht nur wenn der entsprechende Schalter aktiv ist true zurückgeben, sondern auch wenn die Methode isRandomOn() true zurückgibt.
